## 📱 About The App

**LU VIIO UPI Alert** turns your smartphone into a smart UPI soundbox without any additional hardware. Perfect for shopkeepers and small businesses!

### ✨ Features
- ✅ All UPI Apps Support (Paytm, PhonePe, GPay, BHIM)
- ✅ Instant Voice Alerts on Lock Screen
- ✅ Hindi & English Languages
- ✅ No Hardware Needed
- ✅ Works Offline
- ✅ Transaction History
- ✅ Daily/Weekly/Monthly Reports
- ✅ Battery Optimized

## 🎯 How It Works
1. Install the app
2. Grant notification access
3. Start service
4. Get instant voice alerts for every UPI payment!

## 📲 Installation
Download from releases or build from source.
